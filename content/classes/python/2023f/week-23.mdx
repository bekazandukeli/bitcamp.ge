#### სავარჯიშოები

1. გააკეთეთ linked list მონაცემთა სტრუქტურის იმპლემენტაცია პითონში. შექმენით კლასი Node ორი ატრიბუტით: data და next.
შექმენით მეთოდები ახალი ნოუდი ჩასამატებლად დასაწყისში, ბოლოში და კონკრეტულ პოზიციაზე. ასევე განსაზღვრეთ მეთოდი, რომელიც წაშლის ნოუდს კონკრეტულ პოზიციაზე.

2. განსაზღვრეთ Stack მონაცემთა სტრუქტურა პითონში ლისტების გამოყენებით. განსაზღვრეთ მეთოდები: push, pop, peek. დამატებით განსაზღვრეთ მეთოდი, რომელიც შეამოწმებს ცარიელია თუ არა stack-ი.

3. განსაზღვრეთ Queue პითონში ლისტების გამოყენებით. განსაზღვრეთ მეთოდები: enqueue (ელემენტის დასამატებლად), dequeue (ელემენტის ამოსაშლელად) და მეთოდი რომელიც შეამოწმებს არის თუ არა queue ცარიელი.
